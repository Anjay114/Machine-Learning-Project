Data Collection : 1. Data Source : https://archive.ics.uci.edu/dataset/603/in+vehicle+coupon+recommendation
This Dataset consist of 12684 Rows and 25 Columns  
Problem Statement : This project helps to understand whether a person will accept the coupon recommended to him in different driving scenarios
Dataset Information:
This data was collected via a survey on Amazon Mechanical Turk. 
The survey describes different driving scenarios including the destination, current time, weather, passenger, etc., and then ask the person whether they will accept the coupon if they are the driver
